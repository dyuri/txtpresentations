WSGI utility library
by Armin Ronacher (@mitsuhiko)

}}}
{{{ What's included?

* HTTP header parsing
* easy to use Request/Response objects
* URL routing
* form data parsing (file handling included)
* interactive debugger
* session/signed cookie support
* URI/IRI utilities (w/ unicode support)
* test helpers (create environment, request, etc.)
* python 2.6, 2.7, 3.3+, pypy support

Flask is basically Werkzeug + Jinja2 + candy.
